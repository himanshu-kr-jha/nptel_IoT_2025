#### 10. What does the following code do?  
```cpp
int ledPin = 13;
void setup() {
    pinMode(ledPin, OUTPUT);
    for (int i = 0; i < 3; i++) {
        digitalWrite(ledPin, HIGH);
        delay(1000);
        digitalWrite(ledPin, LOW);
        delay(500);
    }
}
void loop() {
    // Do nothing
}
```
Answer: **a) Blink 3 times with 1000ms ON and 500ms OFF**
